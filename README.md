# LT–EN Flashcards

Interactive web-based vocabulary trainer developed using HTML, CSS, and JavaScript.

## Features

- English to Lithuanian flashcards
- Flip card animation
- Previous and Next navigation
- Random word selection
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was developed as a web-based vocabulary learning application to help users practice Lithuanian vocabulary through interactive flashcards.
